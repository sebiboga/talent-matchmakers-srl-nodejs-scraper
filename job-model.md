# Job Model

## Required Fields
- `url` - Job posting URL
- `title` - Job title
- `company` - Company name
- `cif` - Company CIF (8 digits)
- `status` - Job status: scraped, tested, verified, published

## Optional Fields
- `location` - Array of job locations
- `workmode` - Work mode: remote, hybrid, on-site
- `tags` - Array of skills/tags
- `date` - ISO date string
