# Company Model

## Required Fields
- `id` - CIF (8 digits)
- `company` - Full company name

## Optional Fields
- `brand` - Display brand name
- `group` - Company group
- `status` - Company status: activ, suspendat, inactiv, radiat
- `location` - Array of locations
- `website` - Array of website URLs
- `career` - Array of career page URLs
- `lastScraped` - Date in YYYY-MM-DD format
- `scraperFile` - URL to scraper file
